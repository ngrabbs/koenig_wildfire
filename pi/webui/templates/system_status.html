{% if action == "reboot" %}

The Pi is rebooting

Power will cycle and the services will restart automatically. This usually takes about 30–60 seconds.

When you see the wifi reconnect, refresh this page or browse to http://koenig-pi.local:8000 again.

{% else %}

The Pi is shutting down

In a few seconds the Pi will fully power off. You will need to physically unplug and re-plug the USB-C power cable to bring it back up — there is no remote way to turn it on again once it's off.

Wait for any activity LEDs to stop blinking before unplugging.

{% endif %}
